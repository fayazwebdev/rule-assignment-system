
# Rule Assignment System

The objective is to model, store, and present hierarchical rule structures
in a clear and logical way.


## Tech Stack



**Client:** HTML, CSS, JQUERY

**Server:** PHP, MYSQL


## Running Tests

To run tests, run the following command

Clone the repo into XAMPP/htdocs. Import the sql file into MySQL database in XAMPP PHPMYADMIN. Access the url http://localhost/rule-assignment-system/public/index.php

# Database setup
1. Manually you can run from the script exist in the codebase sql/db-script.sql. This will create empty tables.
2. You can also import db with rows from the file sql/rule_assignment_system.sql
